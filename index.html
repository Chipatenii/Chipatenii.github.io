Features

Wall Drawing

Use raycasting to draw walls with distinct colors for different orientations.

Camera Rotation

Rotate the camera using keyboard or mouse controls for an immersive experience.

Player Movement

Move the player with keyboard controls and handle collisions with walls.

About

The inspiration for this project stems from a deep passion for 3D gaming and the desire to explore the capabilities of raycasting technology. Over the past six months, this project has evolved into a comprehensive 3D game development initiative, serving as a Portfolio Project for Holberton School.
